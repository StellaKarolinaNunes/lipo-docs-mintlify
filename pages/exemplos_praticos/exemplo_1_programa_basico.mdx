Neste exemplo, veremos como declarar variáveis, atribuir valores e realizar operações aritméticas básicas na linguagem **LiPo**.

```lipo
// Definindo as variáveis
inteiro a := 10;
inteiro b := 5;

// Soma: adiciona 'a' e 'b'
imprima "Soma: " + (a + b); // Resultado: 15

// Subtração: subtrai 'b' de 'a'
imprima "Subtração: " + (a - b); // Resultado: 5

// Multiplicação: multiplica 'a' por 'b'
imprima "Multiplicação: " + (a * b); // Resultado: 50

// Divisão: divide 'a' por 'b'
imprima "Divisão: " + (a / b); // Resultado: 2
```

---

### O que este código faz?

1.  **Declaração de variáveis**: Usamos `inteiro` para definir que os espaços de memória `a` e `b` armazenarão números sem casas decimais.
2.  **Atribuição**: O operador `:=` (ou `=` dependendo da versão) é usado para atribuir valores às variáveis.
3.  **Saída de Dados**: O comando `imprima` envia o resultado para o console. 
4.  **Operações**: Realizamos as quatro operações matemáticas fundamentais de forma intuitiva.
